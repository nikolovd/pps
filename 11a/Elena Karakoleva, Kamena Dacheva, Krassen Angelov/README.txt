Karatsuba multiplication

Team: Elena Karakoleva, Kamena Dacheva, Krassen Angelov 

The Karatsuba algorithm is a fast multiplication algorithm.
It was discovered by Anatolii Alexeevitch Karatsuba in 1960 and published in 1962.

----------------------------------------------------------
Use function: karatsuba_mul
Example: 
result = karatsuba_mul('arg1','arg2')
- each string (argument) contains one large number
arg1 - first large number 
arg2 - second large number
result - the result of multiplication 
----------------------------------------------------------
